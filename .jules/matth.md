
## 2025-05-24 - Exact Analytical Formulation for Lineage Coupling Expectations
**Learning:** The previous implementation of `calculate_lineage_coupling` used heuristic permutation testing to calculate expected co-occurrences of cell types within clones and their statistical significance. Permutation testing is slow, computationally expensive, and introduces non-deterministic variance that scales with $1/\sqrt{N_{perms}}$. Furthermore, the exact probability distribution for sampling cells without replacement into a clone of a specific size follows a multivariate hypergeometric distribution. We can compute the marginal probability of a clone containing or lacking specific cell types analytically.
**Action:** Replace the heuristic permutation testing with an exact, deterministic analytical formulation. We compute the exact expectation using the inclusion-exclusion principle, where the probability of a clone lacking a cell type is given by the hypergeometric PMF at 0. This is implemented in a fully vectorized manner using the log-gamma formulation (`scipy.special.gammaln`) to avoid factorial overflow and limits. The variance is similarly computed analytically using the sum of independent indicator variables, clamping to zero to avoid numerical precision errors.
