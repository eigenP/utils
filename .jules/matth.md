## 2025-05-24 - Exact Hypergeometric Formulation for Co-occurrence Matrices
**Learning:** Computing the expected random overlap (co-occurrence) between categorical properties (e.g. cell clones and cell types) is typically done via permutation testing, which is slow, non-deterministic, and computationally expensive for large datasets ($O(P \cdot N)$ where P is permutations and N is cells). However, under the null hypothesis that assignments are independent, the presence of a group across multiple categories follows a hypergeometric distribution. By modeling the probability of *absence* ($P(\text{no } j) = \text{hypergeom.pmf}(0, N, K_j, N_c)$) and using the inclusion-exclusion principle, we can directly compute the exact closed-form probability of co-occurrence ($P(j \cap k) = 1 - P(\text{no } j) - P(\text{no } k) + P(\text{no } j \text{ and no } k)$). Because clones are independent, their expected values and variances simply sum linearly.
**Action:** Replace heuristic permutation tests for intersection matrices with the exact analytical hypergeometric formulation. This provides deterministic, exact z-scores and p-values and reduces complexity from $O(P \cdot N)$ to $O(T^2 \cdot S)$ (where $T$ is number of cell types and $S$ is number of unique clone sizes), achieving speedups of >500x.
