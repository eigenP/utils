
## 2025-05-25 - Exact Lineage Coupling via Hypergeometric Distribution
**Learning:** Permutation testing to determine the statistical significance of shared clones across cell types (lineage coupling) is computationally expensive, non-deterministic, and often produces imprecise p-values when the expected intersections are very small. The problem mathematically reduces to drawing cells from a background population without replacement. For a given clone of size $k$ and a cell type $T_1$ of size $N_1$ in a total population $N$, the probability that the clone contains at least one cell of $T_1$ is $1 - P(\text{Hypergeom}(0; N, N_1, k))$. For the intersection of two cell types $T_1$ and $T_2$, we use the inclusion-exclusion principle: $P(T_1 \cap T_2) = 1 - P(\text{no } T_1) - P(\text{no } T_2) + P(\text{no } T_1 \text{ and no } T_2)$, where the last term is exactly $\text{Hypergeom}(0; N, N_1 + N_2, k)$.
**Action:** Replace heuristic, slow permutation tests with the exact, closed-form analytical expectations and variances using `scipy.stats.hypergeom`. This provides deterministic, precise, and significantly faster calculations for $Z$-scores and $p$-values.
