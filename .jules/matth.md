## 2025-04-19 - [Exact NNLS for Regularized Locally Linear Embedding]
**Learning:** The previous implementation of the LLE barycenter heuristic in `kknn_ingest` relied on solving `G * w = 1` analytically and simply clipping the negative values using `np.maximum(w, 0)`. While this avoids negative weights, it is mathematically unsound because simply truncating weights does not produce the optimal Non-Negative Least Squares (NNLS) solution. Redistributing probability mass post-clipping results in a sub-optimal reconstruction under the constraints.
**Action:** The heuristic was replaced with an exact NNLS formulation using Cholesky decomposition (`scipy.linalg.cholesky`, `scipy.linalg.solve_triangular`, and `scipy.optimize.nnls`). This provides the optimal non-negative weights that minimize the reconstruction error, adhering strictly to constraints. Future algorithmic design involving constrained optimization must use exact solvers (when computationally feasible) rather than heuristic approximations like post-hoc clipping.
